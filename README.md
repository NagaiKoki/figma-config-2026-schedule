# figma-config-2026-schedule

A single-file HTML timetable for **Figma Config 2026** sessions — vertical axis = time, horizontal axis = stage. Overlapping sessions sit side by side so you can scan a day at a glance.

## Usage

Just open `index.html` in a browser.

- Click stage chips in the legend to filter venues
- Tap a session to open a bottom sheet with speakers, a paraphrased summary, and the official link
- The **PDF / Print** button in the bottom-right exports the timetable to paper / PDF
- Toggle **EN / 日本語** in the header (default: English)

## Data

- Source: https://config.figma.com/san-francisco/agenda
- All times in **PDT**
- Summaries are paraphrased; refer to the official page for canonical descriptions
